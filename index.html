About The Game

This is an anagram game made by me where each word in a sentence is randomly scrambled (each word is an anagram). Your goal is to unscramble them!

Instructions

Fix the sentence by unscrambling it into the correct order in the input box.

If you're stuck, click "Hint" to get a clue!
